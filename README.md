# Avaliação de Atendimento ⭐

Este é um projeto de front-end que simula uma tela de avaliação de atendimento, com interface moderna e responsiva. O usuário pode selecionar uma nota de 1 a 5 e, ao enviar, recebe uma mensagem de agradecimento com base na escolha feita.

## 🖼️ Prévia

![Prévia do projeto](/image/print.png)

---

## ✨ Funcionalidades

- Escolha de avaliação de 1 a 5
- Destaque visual para a nota selecionada
- Mensagem de agradecimento personalizada
- Interface responsiva (mobile e desktop)
- Estilo moderno e acessível

---

## 💻 Tecnologias utilizadas

- HTML5
- CSS3 (com uso de media queries e customização de fontes)
- JavaScript (DOM + interações)
- SVG para ícone de estrela

---
